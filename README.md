# 📚 Sistema de Biblioteca em Python

Projeto desenvolvido em Python utilizando Programação Orientada a Objetos (POO).

## 🚀 Funcionalidades

* Listar livros
* Emprestar livros
* Devolver livros
* Buscar livros pelo título
* Controle de disponibilidade

## 🛠️ Tecnologias utilizadas

* Python 3

## 📚 Conceitos praticados

* Classes e Objetos
* Métodos
* Atributos
* Encapsulamento básico
* Estruturas de repetição
* Estruturas condicionais
* Listas de objetos

## ▶️ Como executar

```bash
python main.py
```

## 🎯 Objetivo

Praticar os conceitos fundamentais de Programação Orientada a Objetos (POO) em Python através da construção de um sistema simples de gerenciamento de biblioteca.
